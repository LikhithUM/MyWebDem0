2. Unordered List - <ul> (Nested)

This is a nested unordered list inside an item:

3. Ordered List - <ol> (Simple)

This is a basic ordered (numbered) list:

This is a nested ordered list inside an item:

  1. Morning Routine
    1. Wake up
    2. Brush teeth
    3. Exercise
  2. Work Tasks
    1. Check emails
    2. Attend meetings
